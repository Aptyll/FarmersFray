# Farmers Fray
A fast-paced browser MOBA Ã— RTS hybrid built entirely in Vanilla JavaScript.

### Gameplay Design
**MOBA-style victory:** have the last standing bunker to win. 
**RTS control:** select units, place buildings, and buy upgrades.

### Controls
- **Mouse**: 
  - Left-click drag box selection for units and buildings
  - Right-click to command units to move or attack

### UI System
- **6 Action Pages**: Organized hotkey system for quick command access
- **Building Menu**: Access to construction options
- **Upgrade Menu**: Buy Unit and Building Upgrades
- **Real-time Resource Display**: Track resources and supply limits
- **Game Timer**: Monitor match duration
- **Chat System**: Communicate with team or all players
- **Performance Monitor**: Track FPS, frame time, and memory usage
- **Scoreboard**: Live team and player statistics

### Modern & Minimalistic
- Clean, dark-themed interface
- High contrast for important information

### UX-First Approach
- Intuitive controls and feedback
- Clear visual hierarchy
